## 프로젝트 진행상황

### <u>Frontend</u>

#### ✔️ 기능

경기 침체를 대비해 주식, 부동산, 금리 등 다양한 경제 지표를 한눈에 볼 수 있는 웹 애플리케이션 제공

또는, 자산 관리 및 투자에 대한 정보 제공 (사실 지극히 개인적인 목적이기도 함. 그치만 재밌잖아, 미래에 도움도 되고)

이러한 가치가 쌓여 작은 커뮤니티, 새로운 네트워킹 장이 되었으면 좋겠음

- 미 증시 주가 조회

- - 소켓으로 하고픈데 무료 API가 없어 방식 고민중

- 종목별 적정가 조회(운영자 방법론 제시 및 계산기?)

- 부동산, 경제 지식 공유

- 네트워킹이 되는 커뮤니티 방향성이 잡히면, 채팅방 기능 추가적으로

- - 그치만, 일반적인 오픈채팅방, DM 등 기존 방식에서 벗어나 새로운 형태의 SNS를 시장에 던져볼 생각.

#### ✔️ 기술 스택

Framework: Next.js 15.2 (App Router)

State Management: Recoil, React Query

Type Checking: TypeScript

Package Manager: Yarn Berry (PnP, loose mode)

#### ✔️ 디렉토리 구조

```bash
/wetopia
 ├── /public  → 정적 파일
 ├── /src
 │    ├── /app
 │    │    ├── /news
 │    │    │    ├── page.tsx  → 블로그 목록 페이지
 │    │    ├── /page.tsx  → 메인 페이지
 │    │    ├── /layout.tsx  → 공통 레이아웃
 │    ├── /components  → UI 및 공용 컴포넌트
 │    │    ├── /...
 │    ├── /lib  → API, 유틸 함수 ..
 │    ├── /hooks  → 커스텀 훅 (React Query, Recoil 연동)
 │    ├── /styles  → Emotion 스타일
 │    ├── /types  → 타입 인터페이스
 ├── /next.config.js
```

### <u>Backend</u>

#### ✔️ 배포

App Runner 배포 완료

- **런타임**: Node.js 20
- **소스 디렉토리**: `/`
- **빌드 명령어**:
  ```bash
  yarn install && yarn build
  ```
- **시작 명령어**:
  ```bash
  node dist/main.js
  ```
- **포트**: 3000 -> 4000 변경
- **환경 변수 등록 필수**

#### ✔️ 기술 스택

Framework : [Nest.js](https://nestjs.com)

Database : PostgreSQL (Neon.tech에 배포해둔 상태)

ORM: TypeORM

Scheduler: `@nestjs/schedule`

HTTP Client: `@nestjs/axios`

- GraphQL로 바꿀 예정

Hosting: AWS App Runner

#### ✔️ 디렉토리 구조

```bash
wetopia-backend
├── src
│   ├── app.module.ts         # 루트 모듈
│   ├── main.ts               # 앱 진입점
│   ├── config                # 환경변수 설정 모듈
│   │   └── config.module.ts
│   ├── stocks                # 주가 관련 도메인
│   │   ├── dto               # 요청 및 응답 DTO
│   │   ├── entities          # TypeORM 엔티티
│   │   ├── stocks.module.ts
│   │   ├── stocks.service.ts
│   │   ├── stocks.controller.ts
│   ├── external              # 외부 API 연동 (TwelveData)
│   │   └── twelve-data.service.ts
│   ├── common                # 공용 유틸, 필터, 예외 등
│   │   ├── filters
│   │   ├── interceptors
│   │   ├── utils
├── test                     # 테스트 코드
├── .env                     # 환경 변수 파일
├── tsconfig.json
├── package.json
```
