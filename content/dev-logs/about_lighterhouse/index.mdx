## Project Lighter-House

- 배포 완료(lighterhouse.0biglife.com) -> 우측 상단 버튼으로 접속 가능

### <u>Frontend</u>

- Next.js 15.2 App Router + API Route 기반 애플리케이션

- 분석 결과 오버뷰 제공

- 성능 지표별 상세 분석 제공 및 Google 우수-권장 수치화 제공

- Critical/General 우선순위 기반 성능 분석 결과 정렬

- 예상 향상량 가중치 제공 -> 제공하기에 너무 러프한 단계

- 개선항목별 참고 문서 url 연결
