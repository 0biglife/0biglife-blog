## Idea

Two grids are kept on the GPU — one a **velocity field**, one a **dye field**
holding ink color. The cursor stirs the velocity field, and the dye is
carried along by that flow, spreading like ink dropped onto water.

## Core technique

**Semi-Lagrangian advection** — it traces backward, asking "where did this
pixel's color flow in from one frame ago," and samples that point. It is
unconditionally stable, so it never blows up.

```glsl
vec2 vel = texture(u_velocity, v_uv).xy;
vec2 source = v_uv - vel;          // trace one step backward
outColor = texture(u_src, source) * u_dissipation;
```

The velocity field advects itself the same way, and the cursor force plus a
faint curl-noise drift are added in. Four ping-pong passes make up one frame.

## Interaction

Drag the cursor and velocity is carried in that direction while color is
injected. Left alone, an invisible hand slowly bleeds ink. Requires WebGL2
and floating-point render targets.
