## Idea

Near a black hole, not even light travels straight. For every pixel on
screen a single photon is fired, and its path is **integrated
numerically as a geodesic**. The photon is pulled by mass and curves;
once it touches the event horizon it can never escape.

## Core code

Angular momentum `h²` is treated as a conserved quantity, and a
curvature acceleration is added to the photon's velocity each step. The
`r⁻⁵` term produces the sharp bending around the photon sphere.

```glsl
vec3 h = cross(pos, vel);
float h2 = dot(h, h);
// each step:
vec3 accel = -1.5 * h2 * pos / pow(dot(pos, pos), 2.5);
vel += accel * dt;
pos += vel * dt;
```

When the photon crosses the equatorial plane it samples the accretion
disk; if it falls inside the horizon it is painted black. Escaped
photons see starlight along their bent direction.

## Interaction

Drag to orbit the camera around the black hole. Scroll to change
distance. Left alone, it slowly orbits on its own.
