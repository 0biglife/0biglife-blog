## アイデア

ブラックホールの近くでは光さえまっすぐ進めない。画面のすべてのピクセル
から光子を一つ放ち、その経路を**測地線として数値積分**する。光子は
質量に引かれて曲がり、事象の地平線に触れると二度と脱出できない。

## コアコード

角運動量 `h²` を保存量として扱い、毎ステップ光子の速度に曲率加速度を
加える。`r⁻⁵` の項が光子球周辺の急激な曲がりを生む。

```glsl
vec3 h = cross(pos, vel);
float h2 = dot(h, h);
// 毎ステップ:
vec3 accel = -1.5 * h2 * pos / pow(dot(pos, pos), 2.5);
vel += accel * dt;
pos += vel * dt;
```

光子が赤道面を横切ると降着円盤をサンプリングし、地平線の内側に入ると
黒く塗る。脱出した光子は曲がった方向の星明かりを見る。

## インタラクション

ドラッグでカメラがブラックホールを周回する。ホイールで距離を調整。
放置するとゆっくり自動で公転する。
