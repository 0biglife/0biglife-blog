## アイデア

ポリゴンメッシュを使わず、ピクセルごとに光線を放って距離場(SDF)に沿って
進む**レイマーチング**で3D形状を描く。五つの球をなめらかに溶かし合わせ、
ひとつの流れるクロームの塊にした。

## 核心コード

`smin` — 二つの距離場をなめらかにつなぐ平滑最小値の関数が、メタボールの
核心だ。

```glsl
float smin(float a, float b, float k) {
  float h = clamp(0.5 + 0.5 * (b - a) / k, 0.0, 1.0);
  return mix(b, a, h) - k * h * (1.0 - h);
}
```

表面法線で手続き的なスタジオ環境を反射させ、フレネルで縁を明るくして
金属のように見せている。

## インタラクション

カーソル位置がカメラのyaw・pitchになる。マウスを動かすとクロームの塊の
周りを回り、反射が流れる。動かさなければゆっくり自転する。
