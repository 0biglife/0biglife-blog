## Idea

With no polygon mesh, a 3D form is drawn by **ray marching** — shooting a
ray per pixel and stepping it along a distance field (SDF). Five spheres are
smoothly melted together into a single flowing blob of chrome.

## Core code

`smin` — a smooth-minimum function that seamlessly joins two distance
fields — is the heart of the metaball.

```glsl
float smin(float a, float b, float k) {
  float h = clamp(0.5 + 0.5 * (b - a) / k, 0.0, 1.0);
  return mix(b, a, h) - k * h * (1.0 - h);
}
```

The surface normal reflects a procedural studio environment, and a Fresnel
term brightens the edges so it reads as metal.

## Interaction

The cursor position becomes the camera's yaw and pitch. Move the mouse to
orbit the chrome blob as the reflections flow. Left alone, it slowly spins.
