## アイデア

画面全体を覆う一枚の矩形にフラグメントシェーダーをかける。すべての
ピクセルがGPUで同時に計算され、フラクタルノイズをノイズでもう一度
ねじる**ドメインワーピング**の手法で、流れるオーロラをつくる。

## 核心コード

ノイズ座標そのものをノイズの結果で押し出す二段階のワープが核心だ。

```glsl
vec2 q = vec2(fbm(uv + t), fbm(uv - t));
vec2 r = vec2(
  fbm(uv + 4.0 * q + t * 1.4 + mouse),
  fbm(uv + 4.0 * q - t * 1.1 + mouse)
);
float f = fbm(uv + 4.0 * r);
```

値 `f` をコサインパレットに通して虹色を得て、折りたたんだ `sin` で
細い光のリボンを重ねている。

## インタラクション

カーソル位置はノイズドメインのオフセットとして入る。マウスを動かすと
オーロラ全体がそちらへ引かれ、うねる。
