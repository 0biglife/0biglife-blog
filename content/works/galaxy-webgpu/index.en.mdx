## Idea

The stars of a galaxy pull on one another. Each of 4,096 stars
integrates the gravity it feels from **every other star** plus a central
mass, every frame. The N² work (~17 million interactions) — too heavy
for the CPU — is run in parallel on the GPU by a **WebGPU compute
shader**.

## Core code

One compute thread owns one star and sums the force from all the rest.

```wgsl
@compute @workgroup_size(64)
fn main(@builtin(global_invocation_id) gid: vec3<u32>) {
  let i = gid.x;
  var force = vec3<f32>(0.0);
  for (var j: u32 = 0u; j < COUNT; j = j + 1u) {
    let d  = posIn[j].xyz - posIn[i].xyz;
    let r2 = dot(d, d) + P.soft;
    force += d * (P.starMass / (r2 * sqrt(r2)));
  }
  // ...central mass + velocity/position integration
}
```

Position buffers are ping-ponged to avoid read/write conflicts, and the
updated stars are drawn as instanced billboards with additive blending
to make a glowing galaxy.

## Interaction

Drag to orbit the galaxy, scroll to change distance. Left alone, it
slowly turns.

## WebGPU

Browsers without WebGPU support are shown a notice. Runs on recent
Chrome/Edge and Safari 18 or newer.
