## アイデア

銀河の星々は互いに引き合う。4,096個の星それぞれが、**他のすべての星**と
中心質量から受ける重力を毎フレーム積分する。CPU には重い N²(約1,700万)
の計算を、**WebGPU コンピュートシェーダー**が GPU で並列処理する。

## コアコード

コンピュートスレッド一つが星一つを担当し、残りすべてとの力を合算する。

```wgsl
@compute @workgroup_size(64)
fn main(@builtin(global_invocation_id) gid: vec3<u32>) {
  let i = gid.x;
  var force = vec3<f32>(0.0);
  for (var j: u32 = 0u; j < COUNT; j = j + 1u) {
    let d  = posIn[j].xyz - posIn[i].xyz;
    let r2 = dot(d, d) + P.soft;
    force += d * (P.starMass / (r2 * sqrt(r2)));
  }
  // ...中心質量 + 速度・位置の積分
}
```

位置バッファをピンポンで入れ替えて読み書きの衝突を防ぎ、更新された星を
インスタンス化ビルボード + 加算ブレンドで描画して輝く銀河を作る。

## インタラクション

ドラッグで銀河を周回、ホイールで距離を調整。放置するとゆっくり回る。

## WebGPU

WebGPU 非対応のブラウザには案内を表示する。最新の Chrome・Edge、
Safari 18 以降で動作する。
