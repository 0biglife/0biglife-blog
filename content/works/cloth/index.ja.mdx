## アイデア

布は点(質量)と、その間をつなぐ拘束(constraint)の格子だ。各点は
**ベルレ積分**で動き、拘束は毎フレーム長さを元に戻そうとする。限界
以上に伸びた拘束は切れる — それが「裂け」である。

## コアコード

ベルレ積分は速度を保持しない。現在位置と前の位置の差がそのまま速度だ。

```js
const vx = (p.x - p.px) * friction;
const vy = (p.y - p.py) * friction;
p.px = p.x;
p.py = p.y;
p.x += vx;
p.y += vy + gravity * dt * dt;
```

拘束の緩和は二点を元の長さへ引き寄せ、伸びすぎたらリンクを切る。

```js
const d = Math.hypot(dx, dy);
if (d > rest * TEAR) removeConstraint();
else applyCorrection((rest - d) / d);
```

## インタラクション

ドラッグで布の一点をつかんで引っ張る。強く引くと拘束が限界を超えて
裂ける。風が布を揺らす。ダブルクリックでリセット。
