## アイデア

最初は2Dキャンバスに線を描いていたが、今回は同じ点の連なりを
**フラグメントシェーダー**に渡す。すべてのピクセルがGPU上でトレイルの
ポリラインまでの距離を測り、その距離を発光の強さに変えて、やわらかな
光のリボンをつくる。

## 核心コード

JSはカーソルへ補間(lerp)する24点の連なりだけを保持し、距離計算と発光は
すべてシェーダーが担う。

```glsl
float segDist(vec2 p, vec2 a, vec2 b) {
  vec2 pa = p - a, ba = b - a;
  float h = clamp(dot(pa, ba) / dot(ba, ba), 0.0, 1.0);
  return length(pa - ba * h);
}
```

先頭は太く明るく、末尾へ向かうほど細く透明にテーパーさせ、中央の白い
コアと虹色のヘイローでネオンのように光らせている。

## インタラクション

カーソルを止めると、連なりは中央をゆっくり巡るアイドルドリフトに移る。
