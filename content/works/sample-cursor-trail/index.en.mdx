## Idea

It began as lines drawn on a 2D canvas; now the same chain of points is
handed to a **fragment shader**. Every pixel measures its distance to the
trail's polyline on the GPU and turns that distance into glow, forming a
soft ribbon of light.

## Core code

The JS keeps only a 24-point chain that eases (lerps) toward the cursor —
all the distance math and the glow happen in the shader.

```glsl
float segDist(vec2 p, vec2 a, vec2 b) {
  vec2 pa = p - a, ba = b - a;
  float h = clamp(dot(pa, ba) / dot(ba, ba), 0.0, 1.0);
  return length(pa - ba * h);
}
```

The head is thick and bright, tapering thinner and more transparent toward
the tail; a white core plus an iridescent halo make it glow like neon.

## Interaction

When the cursor stops, the chain shifts into an idle drift, slowly orbiting
the center.
