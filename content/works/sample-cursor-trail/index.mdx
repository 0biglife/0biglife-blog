## 아이디어

처음엔 Canvas 2D로 선을 그렸지만, 이번엔 같은 점 사슬을 **프래그먼트
셰이더**에 넘긴다. 모든 픽셀이 GPU에서 트레일 폴리라인까지의 거리를 재고,
그 거리를 발광 세기로 바꿔 부드러운 빛의 리본을 만든다.

## 핵심 코드

JS는 마우스를 향해 보간(lerp)되는 24개 점 사슬만 유지하고, 거리 계산과
발광은 전부 셰이더가 맡는다.

```glsl
float segDist(vec2 p, vec2 a, vec2 b) {
  vec2 pa = p - a, ba = b - a;
  float h = clamp(dot(pa, ba) / dot(ba, ba), 0.0, 1.0);
  return length(pa - ba * h);
}
```

머리 쪽은 굵고 밝게, 꼬리로 갈수록 가늘고 투명하게 테이퍼링하고,
가운데 흰 코어 + 무지갯빛 헤일로를 더해 네온처럼 빛나게 했다.

## 인터랙션

마우스를 멈추면 사슬이 중앙을 천천히 도는 아이들 드리프트로 전환된다.
