## Idea

An invisible vector field (a flow field) is laid across the whole screen,
and 2,800 particles ride its current. Where a particle has passed becomes a
slowly fading trail of light, building river-like patterns.

## Core code

The vector field is built with no noise library — just four layers of sine
waves added together. It keeps the file dependency-free while still
producing a smooth, organic flow.

```js
function fieldAngle(x, y, t) {
  const a =
    Math.sin(x * SCALE + t) +
    Math.cos(y * SCALE * 1.3 - t * 0.8) +
    Math.sin((x + y) * SCALE * 0.7 + t * 0.6) +
    Math.cos((x - y) * SCALE * 1.1 - t * 0.4);
  return a * Math.PI;
}
```

Each frame the background is not fully cleared but painted over slightly
dark (`rgba(...,0.075)`) so the trails dissolve gradually; particle strokes
use `lighter` compositing, so light accumulates where they overlap.

## Interaction

Around the cursor a tangential force is added, creating a vortex. Stir the
mouse and the flow gets sucked in and swirls.
