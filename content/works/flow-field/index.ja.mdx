## アイデア

画面全体に見えないベクトル場(フローフィールド)を敷き、2,800個の粒子を
その流れに乗せる。粒子が通った跡はゆっくり消える光の軌跡となり、川の
ように流れる模様をつくる。

## 核心コード

ベクトル場はノイズライブラリを使わず、サイン波を四層重ねてつくる。
外部依存のない単一ファイルを保ちながら、なめらかで有機的な流れが生まれる。

```js
function fieldAngle(x, y, t) {
  const a =
    Math.sin(x * SCALE + t) +
    Math.cos(y * SCALE * 1.3 - t * 0.8) +
    Math.sin((x + y) * SCALE * 0.7 + t * 0.6) +
    Math.cos((x - y) * SCALE * 1.1 - t * 0.4);
  return a * Math.PI;
}
```

毎フレーム背景を完全には消さず、わずかに暗く上塗りして(`rgba(...,0.075)`)
軌跡を徐々に溶かし、粒子の線は `lighter` 合成で重なるほど光が集まるように
している。

## インタラクション

カーソルの周りには接線方向の力が加わり、渦をつくる。マウスをかき回すと
流れが吸い込まれて渦巻く。
