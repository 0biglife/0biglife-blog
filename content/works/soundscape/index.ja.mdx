## アイデア

録音ファイルではなく、**ブラウザがその場で音を合成する**。五つのボイスが
和音をなし、和音は11秒ごとにゆっくり変わり、その上にメロディの音が
ときおり落ちる。ディレイのフィードバックが空間をつくる。

同時に `AnalyserNode` がその音をFFTで分析し — 画面が*いま聞こえている音
そのもの*に反応する。低音は大きな脈動、高音は散らばる星。

## 核心テクニック

Web Audio APIのノードグラフ — オシレーター → フィルター → ディレイ → 分析器。

```js
osc → voiceGain → lowpassFilter → master → analyser → 出力
                              ↘ delay → feedback ↺
```

## インタラクション

- **カーソル左右** — ローパスフィルターのカットオフ。左は暗く丸く、
  右は明るく澄む。
- **カーソル上下** — 落ちてくるメロディ音の密度。上ほど密になる。

ブラウザの自動再生ポリシーのため、最初に一度だけ画面を押して音を
オンにする必要がある。
